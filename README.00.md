Технологии:

* **Apache Hadoop** — это платформа (экосистема) для распределенного хранения и обработки больших данных
* **Apache NiFi** — это инструмент для работы с данными, потоковой интеграции и обработки. с визуальным подходом, вместо написания кода
* **Apache Airflow** — это инструмент для оркестрации (планирования и управления), который управляет и автоматизирует сложные рабочие процессы ETL и DAG
* **Apache Spark** — это фреймворк для распределённой обработки больших данных, для пакетной (batch) обработки и аналитики
* **Apache Flink** — это фреймворк для распределённой обработки больших данных, для реального времени с низкой задержкой
* **Apache Iceberg** — это высокопроизводительный формат таблиц для больших аналитических наборов данных в озерах данных, работает через манифест-файлы (Parquet или ORC), имеет атомарные транзакции и обеспечивает надежное управление изменениями
* **Apache Hive** — это SQL-инструменты для работы с большими данными, для пакетной (batch) обработки, высокие задержки, не для OLTP, но с отказоустойчивостью, т.к. хранит результаты
* **Apache Impala** — это SQL-инструменты для работы с большими данными, для реального времени (MPP - массовая параллельная обработка) с низкими задержками, аналитические запросы, но не отказоустойчивый, т.к. не сохраняет результаты
* **Apache Zeppelin** — это интерактивная веб-оболочка для анализа и визуализации данных, позволяет объединять код (частности Python), текст и визуализации в одном документе. используется с экосистемой Hadoop и Spark, а аналог Jupyter - более гибкий, больше инструментов, но менее заточен под большие данные и экосистему Hadoop
* **Apache Superset** — это платформа для исследования и визуализации больших данных, позволяющая создавать интерактивные дашборды
* **Database tools** — это комплекс ПО и утилит для создания, управления, извлечения, редактирования, трансформации и анализа данных, хранящихся в базах данных. используются в процессах ELT (T - transform)
* **Tableau** — это программное обеспечение для бизнес-аналитики (BI) и визуализации данных
* **Parquet** – это бинарный формат хранения больших данных, ориентированный на столбцы (колонки), который обеспечивает высокую эффективность чтения и сжатия данных. изначально для экосистемы Apache Hadoop, он также широко используется со Spark, Kafka и ClickHouse
* **Pandas** — это библиотека Python для работы с данными, предоставляющая мощные структуры и инструменты для их обработки, анализа и очистки

Словарь:

* Имбалл — это сокращение от «Кимбалл» (Kimball), имени Билла Кимбалла, известного своей работой над моделированием хранилищ данных, в частности, с использованием звездообразной схемы
* Инмон — это имя Билла Инмона, который разработал подход к созданию централизованного корпоративного хранилища данных
* Data Vault — это методология моделирования данных, ориентированная на гибкость, адаптивность и отслеживание истории, состоящая из «хабов», «линков» и «сателлитов»
* Anchor Modeling — это гибкая методология проектирования хранилищ данных, предназначенная для быстрого изменения структуры и масштабирования
